# sdyz库使用说明

## 一、项目介绍
**sdyz**是**三点一洲工作室**的标准命名空间，包含了如下几个类库：

- **aes:** aes密码工具

- **arg_parser:** 用于管理命令行参数

- **cast:** 类型转换工具

- **console:** 控制台工具

- **des:** des密码工具

- **exception:** sdyz库的标准异常类

- **hash:** hash算法工具

- **plus:** 包含了一些字符串操作以及文件的操作

- **tools:** 一些其余的工具

- **vector:** 一些容器工具

**（其中aes、des、sha1的核心算法代码来自于网络）**
